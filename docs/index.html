Rust 烹饪书

Rust 烹饪书

这个Rust 烹饪书是一组简单的例子，演示了如何使用 Rust 生态系统的箱子，完成常见编程任务的良好实践。

了解 Rust 烹饪书 更多，包括如何阅读这本书的提示、如何使用示例，以及约定好的注释风格。

贡献

这个项目的目的是，让新的 Rust 程序员能够容易做出贡献，帮个小忙，并且是一个与 Rust 社区接触的简单方法。本库十分欢迎帮助。详情见CONTRIBUTING.md.
